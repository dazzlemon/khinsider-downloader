Usage:

1. download source
2. install python, scrapy, and parfive
3. Run run-scripts:
   
    Linux:
    ```sh
    run.sh albums.txt [<ARGUMENTS FOR SCRAPY>]
    ```
    
    Windows:
    ```sh
    run.bat albums.txt [<ARGUMENTS FOR SCRAPY>]
    ```

    the most useful argument is `-s FILES_STORE=<PATH TO WHERE YOU WANT ALBUM DOWNLOADED TO>`
